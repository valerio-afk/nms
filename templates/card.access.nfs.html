Connecting using NFS

  1. Open the Terminal
  2. Install nfs client if it's not already in there. Here there are some directions to it on major distributions:
    • Ubuntu/Debian: $ sudo apt install nfs-common
    • Fedora/Red Hat/CentOS/AlmaLinux: $ sudo dnf install nfs-utils
    • ArchLinux: $ sudo pacman -S nfs-utils
    • OpenSUSE: $ sudo zypper install nfs-client
  3. Create a mount directory in either /media or /mnt. For example
    $ sudo mkdir /media/nms
  4. Type the following command
    $ sudo mount -t nfs IP:{{ mountpoint }} /media/nms
  5. You should replace IP with the IP address you can find in the Network tab.
  6. Adapt /media/nms as per step (3) if you used a different setup.

  1. Open the Terminal
  2. Create a mount directory in /Volumes. For example
    $ sudo mkdir /Volumes/nms
  3. Type the following command
    $ sudo mount -t nfs -o resvport IP:{{ mountpoint }} /Volumes/nms
  4. You should replace IP with the IP address you can find in the Network tab.
  5. Adapt /Volumes/nms as per step (2) if you used a different setup.

  1. Go to Control Panel > Programs
  2. On the left side, press Turn Windows features on or off.
  3. Make sure that Services for NFS is checked. Then press OK.
  4. Open Powershell.
  5. Type the following command
    mount -o anon \\IP\{{ mountpoint.replace("/","\\") }} Z:
  6. You should replace IP with the IP address you can find in the Network tab.
  7. You can choose a different unit letter than Z:.
