Connecting using FTP

  1. Download an FTP Client. There are many available with pros and cons. If you are not sure what to use, download FileZilla, as it's cross-platform.
  2. Insert the IP of this NAS (you can find in the Network tab) in Host.
  3. The username and password to use are the same as you set for the SSH service (SSH service is not required to be enabled, but they share the same username).
  4. Leave the Port either blank or the default value you find.
  5. Press Quickconnect (or equivalent for your FTP client).
  6. Transfer files inside {{ mountpoint }}
