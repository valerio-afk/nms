Connecting using SSH

  • Open the Terminal
  • Type the following command
    $ ssh {{ form.username.default }}@IP -p {{ form.port.default }}
  • You should replace IP with the IP address you can find in the Network tab.
  • The first time you connect, please accept the fingerprint by typing yes.
  • Enter the password when asked.

  • Download PuTTY.
  • Once you have opened PuTTY, select SSH in Connection type.
  • Insert the IP of this NAS (you can find in the Network tab) in Hostname.
  • Change Port with {{ form.username.default }}
  • Press Open
  • You will be asked for your login (username) and password.
