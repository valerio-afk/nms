Connecting using SMB

  1. Open Files.
  2. Go to Other locations.
  3. Go to the textbox at the bottom saying Enter server address...
  4. Type smb://IP/NMS. Replace IP with the IP address/hostname of this NAS (you can find in the Network tab).
  5. Press Connect.
  6. When prompted, selected Registered user.
  7. Type {{ form.username.default }} and the password you have set in the credential textboxes.
  8. Press Connect.

  1. Open the Terminal
  2. Install the Common Internet File System (CIFS) utilities if they are not already in there. Here there are some directions to it on major distributions:
    • Ubuntu/Debian: $ sudo apt install cifs-utils
    • Fedora/Red Hat/CentOS/AlmaLinux: $ sudo dnf install cifs-utils
    • ArchLinux: $ sudo pacman -S cifs-utils
    • OpenSUSE: $ sudo zypper install cifs-utils
  3. Create a mount directory in either /media or /mnt. For example
    $ sudo mkdir /media/nms
  4. Type the following command
    $ sudo mount -t cifs //SERVER/IP/NMS /media/nms -o username={{ form.username.default }}
  5. You should replace IP with the IP address you can find in the Network tab.
  6. Type your password once asked.

  1. Open Finder.
  2. In the menu on top, press Go > Connect to Server.
  3. From now on, you can follow similar steps as in Ubuntu starting from step (4).

  1. Open File Manager
  2. In the textbox for the path, insert \\IP\NMS. Replace IP with the IP address/hostname of this NAS (you can find in the Network tab). Then, press Enter key.
  3. When prompted, insert your credential.
  4. Type {{ form.username.default }} and the password you have set.
  5. Press Connect.
